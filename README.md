# zerm

An Eleventy port of the original **zerm** Zola theme.

Original project: <https://github.com/ejmg/zerm>

## What this repo is

This repository preserves the original zerm visual style while converting the implementation to:

- Eleventy 3.1.5
- Nunjucks templates
- Luxon date handling
- Pagefind search
- Atom, RSS, JSON Feed, and twtxt outputs

## Development

```bash
npm install
npm run start
```

## Production build

```bash
npm run build
```
